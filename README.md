# caperone Enterprise store 
Project overview
Caperone Enterprises is a static, creative dropshipping e-commerce website that showcases curated products (Jewels, Shoes, Phones, Wigs etc) and provides a simple online ordering flow and contact channels. The site is designed desktop-first with generous margins and framed content, distinct color themes per page, product cards with images and badges, client-side search and filters, quick-view modals, and a pre-fillable order form that forwards orders to your email.

Key features
Static multi-page site: Home, Products, Order, Contact

Desktop-friendly boxed layout with increased margins and footer pinned below content

Distinct color accents per page (gold, teal, purple, coral) for visual separation

Product grid with images, price, badges (New, Bestseller) and category metadata

Client-side search and category filters (no backend required)

Quick-view modal for product previews and one-click order links

Order form that pre-fills product name via query string and submits to email (FormSubmit)

Contact form and click-to-call / mailto links using provided phone and email

Accessible markup and responsive breakpoints for smaller screens

Lightweight JS for product rendering, filtering, modal, and pre-fill behavior

Tech stack
HTML5 (static pages)

CSS3 (desktop-first responsive styles; single stylesheet)

Vanilla JavaScript (client-side product data, UI interactions)

Form handling via FormSubmit (static form to email)

Static hosting friendly: GitHub Pages, Netlify, Vercel

Repo structure
index.html — Homepage

products.html — Products listing, search, filters, quick-view modal

order.html — Order form (pre-fill support)

contact.html — Contact page and contact form

css/style.css — Centralized styling and desktop layout rules

js/main.js — Product dataset and client-side interaction logic

assets/images/ — Product images (placeholders or real images)
